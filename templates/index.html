THE FINDER





The Finder project


“The Finder” is a client-server platform that aim to detect near-duplicate images through two different search algorithms. The first one consists in searching in a large dataset all the images that are near-duplicates of an uploaded image (useful for detection), the second one selects all the groups of near-duplicate images within an uploaded folder (useful for storage optimization). The proposed result will be the weighted sum of three methods: histograms comparison, features comparison, ORB.
